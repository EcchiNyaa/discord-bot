# EcchiNyaa Bot

Work in progress.

1. Faça uma cópia do arquivo `config/config-default.yml` para `config/config.yml`.
2. Certifique-se de que os módulos estão incluídos em `nyaa.db`.
3. Instale dependências `bundler`.
4. Execute o bot `ruby nyaa.rb`.
